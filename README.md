# Rocket Reactions

  [Demo](https://rocket-reactions.netlify.app/).

## Installation
* `git clone git@github.com:sivanarayanan93/rocket-reactions.git` this repository
* `cd rocket-reactions`
* `yarn install`

## Running in local
  `yarn start`
   Open [http://localhost:3000](http://localhost:3000)
## Production build
  `yarn build`
